## 🚀 Nova — язык программирования для эпохи AI

> **«LLM пишет код — человек ревьюит».**
> Первый язык, спроектированный под эту эпоху.

Алгебраические эффекты делают побочные действия **видимыми прямо в сигнатуре функции**.
Прочитав одну строку, ревьюер понимает: ходит ли код в БД, в сеть, читает ли время, может ли упасть — **без чтения тела функции**.
Это меняет правила игры для ревью AI-кода.

```nova
fn process_order(o Order) Db Net Time Fail -> Receipt
```

Из сигнатуры сразу видно: функция работает с **БД**, **сетью**, читает **время**, **может упасть** — и больше ничего.

### Что делает Nova особенным

- ⚡ **Эффекты + хендлеры** — тесты без моков, ровно тот же код в проде и в тестах
- 🔀 **Без `async`/`await`** — структурированная конкуренция, нет «цвета функций»
- 🛡️ **Контракты — градиент от Go до F\*** — от скрипта до verified-кода в одном языке
- 🎯 **Capability security** — `forbid Net, Fs` запрещает эффект на уровне компиляции
- ⏱️ **Realtime-блоки** — компилятор гарантирует отсутствие GC-пауз для аудио/трейдинга/embedded
- 🔄 **Time-travel debugging** — естественное следствие хендлеров
